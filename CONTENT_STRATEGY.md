## 3. AI-Search Optimization (SGE / ChatGPT / Perplexity)

### Principles

AI search engines pull answers from pages that:
1. **Directly answer the question** in the first 2–3 sentences
2. Use structured, scannable format (bullets, tables, numbered steps)
3. Include **FAQ schema** (`FAQPage` + `Question`/`Answer`)
4. Are authoritative (backlinks, reviews, structured data)

### Pages with FAQ schema (priority)

- `index.html` — ✅ Has FAQPage schema (6 questions)
- Each activity page should have 3–4 FAQ schema entries
- Blog posts: add FAQ schema at the end of each post

### Answer box targets (direct answer format)

Write these answers in posts, first paragraph, 40–60 words:

| Question | Target page |
|----------|------------|
| "How long does it take to learn kitesurfing?" | `how-long-to-learn.html` |
| "Best time to kitesurf in Koh Phangan?" | `best-time-koh-phangan.html` |
| "How much do kitesurfing lessons cost in Thailand?" | homepage + lesson pages |
| "Is Koh Phangan good for learning kitesurfing?" | `beginner-guide-koh-phangan.html` |
| "What equipment is included in kitesurfing lessons?" | `what-equipment-is-included.html` |

### Conversational content structure

Every blog post should open with a direct answer:
```
[Target question] — [Direct answer in 1–2 sentences].
[Supporting detail]. [CTA or transition to full article].
```

Example for "how-long-to-learn.html":
> "Most beginners can ride independently after 6–10 hours of instruction — typically 3–5 days at Kite Club Koh Phangan. The Discovery Course (2h) covers the basics; the Full Beginner Course (6h) takes you to water starts; the Independent Rider Course (10h) gets you riding upwind on your own."

## 4. Internal Linking Map

### Hub-and-spoke structure

```
Homepage
├── /kitesurfing/          → /kitesurfing/lessons/  → each lesson page
│                          → /kitesurfing/koh-phangan/
├── /wing-foil/            → /wing-foil/lessons/    → each lesson page
├── /windsurfing/          → /windsurfing/lessons/  → each lesson page
├── /efoil/                → /efoil/lessons/        → each lesson page
├── /kayaking/
├── /sup-rental/
├── /kite-safari/
├── /school/               → /school/kitesurfing/, /school/wing-foil/, etc.
├── /blog/                 → each blog post
├── /locations/
├── /about/
└── /contacts/
```

### Required internal links per page type

**Activity pages** (kitesurfing, wing-foil, etc.) must link to:
- Their lessons index page (`/kitesurfing/lessons/`)
- `/contacts/` or WhatsApp CTA
- 1–2 related blog posts

**Blog posts** must link to:
- The most relevant activity page (e.g., kitesurfing blog → `/kitesurfing/`)
- The most relevant lessons page if discussing learning
- `/contacts/` CTA at the end

**Lesson pages** must link to:
- Parent activity page
- `/contacts/` or WhatsApp booking
- Sibling lesson pages (e.g., Discovery → Full Beginner → Independent)

### Priority internal links missing (audit needed)

- Blog posts likely lack links back to service pages
- `/school/` sub-pages may not cross-link to activity pages
- Individual lesson pages may not link to each other
